# Chrome Extension: E-commerce Price History and Deal Tracker

A Chrome extension for online shoppers, resellers, and procurement teams that tracks price history for any product across Amazon, Walmart, Best Buy, and 50 other major retailers. Displays 90-day and 365-day price charts directly on product pages, alerts users when watched items drop below target prices, detects fake discount inflation, and exports watchlist data as CSV for purchasing departments.

## Installation

### Chrome
1. Download and extract the ZIP
2. Go to `chrome://extensions/`
3. Enable "Developer mode"
4. Click "Load unpacked" and select the extracted folder

### Firefox
1. Download the Firefox version
2. Go to `about:debugging`
3. Click "Load Temporary Add-on"
